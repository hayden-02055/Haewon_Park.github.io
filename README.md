# Portfolio - 박해원

Backend Engineer · System Architect 포트폴리오 웹사이트

## Tech Stack

- React 18
- Tailwind CSS 3
- Vite 5

## 로컬 개발

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

## 빌드 & 배포

```bash
# 프로덕션 빌드
npm run build

# 빌드 결과 미리보기
npm run preview

# GitHub Pages 배포
npm run deploy
```

## GitHub Pages 배포 설정

1. GitHub 리포지토리 Settings > Pages
2. Source를 `gh-pages` 브랜치로 설정
3. `npm run deploy` 실행

또는 GitHub Actions를 사용하여 자동 배포할 수 있습니다.
